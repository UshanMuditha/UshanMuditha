## 🔬 Projects

### ⚡ [Guardian Drive — Intelligent Safety Car Controller](https://ushanmuditha.github.io/project-guardian.html)
> Sensor-integrated vehicle safety system preventing drunk driving, monitoring driver fatigue via eye blink detection, and collision avoidance using IR sensors. Real-time acquisition via LabVIEW & NI-DAQ.

`LabVIEW` `NI-DAQ` `IR Sensors` `Feedback Control` `Safety Systems`

---

### 💊 [MediBox — Smart Pharmaceutical Storage](https://ushanmuditha.github.io/project-medibox.html)
> IoT-based intelligent medicine storage with temperature/humidity monitoring, alarm management, light regulation, and a Node-RED dashboard — built on ESP32 with MQTT communication.

`ESP32` `MQTT` `Node-RED` `IoT` `C++`

---

### 🌿 [Automated Weed Removal System](https://ushanmuditha.github.io/project-weed.html)
> Laser-equipped gantry robot prototype for chemical-free precision weeding. 3D modelled in SolidWorks; trajectory design and full dynamic simulation in MATLAB & Simulink.

`MATLAB` `Simulink` `SolidWorks` `Robot Kinematics` `PID Control`
